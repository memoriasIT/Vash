# Command-Line

+--------------------+
| @user ~            |
| cd test/           |
|                    |
|                    |
|                    |
| ---                |
+--------------------+

- Git integration
- Functional programming
- Pipes
- Exec and print

- Average user
'''
    @user ~
    cd test/
'''


- Root as bold
'''
    @r00t ~
    cd test/
'''


- User with git
'''
    @user - master ~
    cd test/
'''
